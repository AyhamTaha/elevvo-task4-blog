# 📝 Personal Blog Homepage – Task 4

A responsive personal blog homepage built using **React + Tailwind CSS** as part of my Frontend Internship Task.

---

## 🚀 Features

- 📌 Clean Header with Blog Title
- 🔎 Search posts by title keyword
- 🗂 Filter posts by category (Tech, Travel, Food)
- 📄 Pagination (limit number of posts per page)
- 📱 Fully Responsive Grid Layout
- 🎨 Modern UI with Tailwind CSS
- ⚡ Conditional Rendering (No posts found message)

---

## 🛠 Tech Stack

- React (useState, Props)
- Tailwind CSS
- JavaScript (ES6)
- Vite
